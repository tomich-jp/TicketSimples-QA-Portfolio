# 🎫 TicketSimples - Quality Assurance Portfolio

Este repositório apresenta a estratégia completa de Garantia de Qualidade para o sistema **TicketSimples**, desenvolvido durante o 5º semestre de ADS na **Universidade São Francisco (USF)** e aprimorado com técnicas de Engenharia de QA da **EBAC**.

## 🚀 Objetivo
Validar o fluxo de suporte de TI, garantindo que a transição de canais informais para um sistema centralizado ocorra com integridade de dados e segurança.

## 🛠️ Tecnologias Envolvidas
- **Aplicação:** PHP 8.2, MySQL 8.0, Bootstrap 5.
- **Metodologia:** BDD (Behavior Driven Development) e Testes de Caixa Preta.
